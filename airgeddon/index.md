 ### airgeddon
 
  airgeddon is a menu driven 3rd party tools wrapper to audit wireless networks
  with many features.
 
 **Installed size:** `4.14 MB`  
 **How to install:** `sudo apt install airgeddon`  
 
 {{< spoiler "Dependencies:" >}}
 * aircrack-ng
 * bash 
 * gawk
 * iproute2
 * iw
 * pciutils
 * procps
 * tmux
 * xterm
 {{< /spoiler >}}
